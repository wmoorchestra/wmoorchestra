# WMO Orchestra

This repository contains the source materials for the **WMO Orchestra** project website and accompanying documentation.

It is intended as a minimal public record of the project rather than a development archive or codebase.


## What This Repository Is Not

* This is **not** an open-source project.
* This is **not** a development log.
* This is **not** a music distribution mechanism.
* This is **not** a theory or framework repository.

The repository exists to support publication, archiving, and reference.
